# Dictionary API Documentation
## Overview
The Dictionary API offers a simple and efficient way to access word definitions, synonyms, antonyms, and parts of speech. It is designed to be user-friendly and easy to integrate into your applications.

## Endpoints
### 1. Get Word Definition
- **Endpoint**: `/api/dictionary/{word}`
- **Method**: `GET`
- **Parameters**:
  - `word` (string, required): The word you want to look up.
- **Response**:
    - `definition` (string): The definition of the word.
    - `partOfSpeech` (string): The part of speech of the word (e.g., noun, verb).
    - `examples` (array): An array of example sentences (3) using the word.



## Example

To see the Dictionary API in action, you can use the following example. This example demonstrates how to fetch the definition of a word from the API

```javascript
//make a req to the api lol
fetch("https://dictionary.fonders.org/dictionary/hello")
  .then(response => {
    if (!response.ok) {
      throw new Error(`Error: ${response.status}`);
    }
    return response.json();
  })
  .then(data => {
    console.log(data);
  })
  .catch(error => {
    console.error(error);
  });
```
